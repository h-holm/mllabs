HOW TO RUN
==========

requirements:
    - python3 (I used 3.7.4)
    - virtualenv

Step 1: Create environment

    virtualenv -p python env

Step 2: Install requirements

    pip3 install -r requirements.txt

Step 3: Run

    source venv/bin/activate

Step 4: Deactivate virtual environement

    deactivate
